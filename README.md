# Волонтерское бюро
## Пояснительная записка
Авторы: Ермиличева Алёна, Кошелева Анна, Гаврилов Александр
### Описание проекта:
Волонтерское бюро. Сайт состоит из ленты афиш добрых дел (волонтерство: уборка, высадка лесов, корм животных, уборка улиц, субботники и тп), на которые авторизованный пользователь может записаться. Начальный экран - лента с объявлениями. Любой пользователь может просматривать их. Доступна кнопка «профиль», благодаря которой, пользователь может зарегистрироваться/авторизоваться. Авторизовываясь, у пользователя появляется возможность выкладывать объявления и откликаться на другие. Афиши будут фильтроваться по местоположению пользователя/городу, который введет пользователь.
